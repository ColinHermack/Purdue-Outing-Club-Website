This site was built with NextJS and styled with TailwindCSS. It is based on the NextJS Portfolio with Blog template
by Vercel. It uses the Geist Sans typeface. This site has no backend. All data handling is done with Markdown and JSON.
It is deployed with Vercel.

## Tech Stack
ReactJS, NextJS, TailwindCSS, Vercel
